Remind Mail – Administer Ref - V1


To update reminder information in a file follow the below set of steps :


Req OS : Linux


  1. Start Terminal


  1. Download Private Key file – NaggyKeyPair.pem to a directory ( for eg: ~/Desktop/Suganya/aws/imp)

scp -i NaggyKeyPair.pem ec2-54-211-220-96.compute-1.amazonaws.com:~/NaggyKeyPair.pem ~/Desktop/Suganya/aws/imp/NaggyKeyPair.pem

  1. Type or Copy below command ( Ctrl + C)

ssh -i ~/Desktop/Suganya/aws/imp/NaggyKeyPair.pem ec2-user@ec2-54-211-220-96.compute-1.amazonaws.com

3.1) If you did copy , paste it in terminal ( Ctrl + Shift + V )


4. To go to the directory where you have the reminder info file

cd /var/www/html


5. To open the file in edit mode

vi reminderinfo.csv

6. Press I to insert


7. To add new mail id info , follow the below format

Eg ) mail to| suganya.rahulkanna@gmail.com

Note :

> This is like a header for the reminder information that you are going to add

> It can be followed by yearly, monthly, daily mng, daily evng reminders

> You can go to the bottom most portion of the file , when you add your mail info and reminder info for the first time , also you can have a empty line below you give your mail id info.


8. Format of yearly reminder:

yearly|Month either in number or in 3 chars|date in 1 or 2 digits|reminder information

eg) yearly|Jun|28|Rahul kanna's b'day


9. Format of monthly reminder:

monthly|date in 1 or 2 digits|reminder information

eg) monthly|31|Pay house rent|Pay Elec Bill|Pay Vonage Bill


10. Format of daily reminder which u like to remind you in morning at 10 pm.

Fmt : daily mng| reminder info

eg: daily mng| Don't forget to take Phone , Car & House key


11. Format of daily reminder which u like to remind you in evng at 6 pm.

Fmt : daily evng| reminder info

eg: daily evng| Don't forget to bring lunch box

Type :wq – to save and quit the file.
